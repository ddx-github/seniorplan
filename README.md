## 📋 计划表结构说明

本计划表为**单工作表格式**，包含以下六大区域：

### 一、每日时间分配（约7-7.5小时）

| 时段 | 科目 | 时长 | 内容概要 |
|------|------|------|----------|
| 上午 8:30-11:30 | 数学 + 物理 | 3小时 | 暑假作业 + 视频 + 知识点梳理 |
| 下午 14:00-16:30 | 语文 + 英语 | 2.5小时 | 暑假作业 + 文言文/视频 + 词汇 |
| 晚上 19:30-20:00 | 化学 | 30分钟 | 暑假作业 + 知识点 + 练习（每天必做） |
| 晚上 20:00-20:30 | 生/政/史/地 | 30分钟 | 四科轮换：生物→政治→历史→地理 |
| 晚上 20:30-21:00 | 阅读 | 30分钟 | 《乡土中国》+ 自选阅读书目（每天必读） |

**每周安排**：学习6天（周一到周六），周日自由安排。

### 二、每周详细安排
具体到每个时段的科目和内容，含两次15分钟休息时间。

### 三、三阶段航行计划

#### 🌙 播种期（7.20-8.10，共22天）
- **目标**：完成暑假作业主体，把新知识的轮廓描摹在脑海里
- **各科重点**：
  - 数学：完成初中衔接（公式+函数图象），开始高中集合与函数
  - 物理：完成《运动的描述》8道检测题
  - 语文：读完《乡土中国》第1-5篇并写旁批；翻译背诵《劝学》；完成现代文阅读
  - 英语：完成语法测试+七选五+语法填空；每天背20-30词
  - 化学：默写1-20号元素+原子结构+化合价+离子+化学式+俗名+溶解性
  - 生物/政治/历史/地理：完成暑假作业主体部分
  - 阅读：优先读完《乡土中国》，开始文学类自选书目

#### 🌟 浇灌期（8.11-8.25，共15天）
- **目标**：让模糊的影子变清晰，阅读进入深水区
- **各科重点**：
  - 数学：攻克高中函数单调性与最值，回顾错题
  - 物理：力学综合题尝试，牛顿运动定律视频再听一遍
  - 语文：《高中必背文言文默写》每天1-2篇；继续《乡土中国》6-14篇
  - 英语：读《书虫》系列；背《新概念英语2》第1-25篇
  - 化学：预习物质的量+物质分类+离子反应+氧化还原
  - 生物：完成B组新高考模拟，了解高考题型风格
  - 政史地：回顾错题，画思维导图；配合纪录片学习
  - 阅读：读人文类+科学类自选书目

#### ☀️ 收束期（8.26-8.31，共6天）
- **目标**：整理行囊，迎接新的黎明
- **重点任务**：
  - 检查9科暑假作业是否全部完成
  - 整理笔记和错题本
  - 回顾《乡土中国》旁批，整理阅读心得
  - 调整作息（早7点起，晚10点半休息）
  - 准备开学用品（必备书籍+笔记本+文具）

### 四、暑期阅读计划

**阅读书目来源**：
- 《高中生基础阅读书目》（人民日报推荐，共30本）
- 2本必读名著（《乡土中国》+工具书类）
- **合计32本，建议精选8-10本深入阅读**

**推荐优先级**：
1. **必读**：《乡土中国》（费孝通）— 语文作业要求
2. **文学类**：《红楼梦》《呐喊彷徨故事新编》《围城》
3. **人文类**：《中国哲学简史》（冯友兰）、《万历十五年》（黄仁宇）
4. **科学类**：《从一到无穷大》（伽莫夫）、《科学的历程》（吴国盛）

> 每天阅读30分钟，43天累计约21.5小时，足够读完5-8本书。其余书目可开学后慢慢品读。

### 五、预习日历视图
7月20日至8月31日的完整日历，用颜色标注三个阶段和周日休息。建议在完成当天任务后在格子里画★标记。

## 📚 暑假作业整合说明

本计划已整合《初高衔接暑假作业》全部9科内容，作业顺序为：

**语 → 数 → 英 → 物 → 化 → 生 → 政 → 史 → 地**

| 科目 | 暑假作业内容摘要 |
|------|------------------|
| **语文** | 《乡土中国》阅读+旁批（前5篇/全14篇）、《劝学》翻译背诵默写、现代文阅读《割草》、名篇名句默写10题 |
| **数学** | 初中衔接：公式展开、分母有理化、立方和差、因式分解、一元二次方程、二次函数最值、函数图像 |
| **英语** | 语法测试15题、七选五、语法填空（10题）、建议看英文电影/读《书虫》/背《新概念英语2》/拍V-log |
| **物理** | 《运动的描述》8道检测题（质点、参考系、位移路程、时间时刻、加速度、v-t图像等） |
| **化学** | 熟记1-20号元素、原子结构示意图、化合价口诀、常见阴阳离子、化学式、俗名、溶解性口诀 |
| **生物** | A组基础衔接（3题）+ B组新高考模拟（NTUs植物源光合系统，4小题） |
| **政治** | 初中回顾（国家权力机关、分配方式、改革开放成就）+ 高中预习（人类社会存在基础、科学社会主义诞生、社会形态连线） |
| **历史** | 预习《中外历史纲要》（上册），重点：统一多民族国家形成与发展、政治制度演变、经济发展；推荐纪录片《中国通史》 |
| **地理** | 准备立体地图+地球仪；完成世界气候分布图填空涂色；完成中国地形图填空（山脉+四大盆地/高原+三大平原/丘陵）；推荐纪录片6部 |

## 🎯 使用建议

1. **打印后贴在书桌前**，每天对照执行，用铅笔在日历上标记完成度。
2. **优先保证暑假作业完成**，这是开学要交的任务，预习是锦上添花。
3. **《乡土中国》是重中之重**，务必在8月10日前读完前5篇并写好旁批。
4. **数学初中衔接内容**（公式/函数/方程）是高中地基，务必在播种期夯实。
5. **化学元素和化合价**是"字母表"，建议每天花10分钟默写，分散记忆比突击有效。
6. **阅读不必贪多**，32本书选8-10本精读即可，其余可列入开学后的长期书单。
7. **周日是缓冲带**，可以补作业、多读书、或者去外面走走，不要给自己太大压力。
8. **如果某天累了**，就少学一点，但不要完全停下——流水不争先，争的是滔滔不绝。
